# Release Notes v1.0.0

This release completely revamps the repository into a highly curated, English-only job board and resource list focused on IT workers based in Australia, with specific emphasis on remote, Perth-friendly, and contractor roles.

## Key Highlights

- **English Translation & Curation:** Removed non-English job boards and translated all repository documentation into clean English.
- **Australia-Focused Curation:** Shifted the focus specifically to Australia, APAC, and compatible remote roles.
- **IT-Specific Filtering:** Removed unrelated industry boards to focus purely on IT Support, Linux SysAdmin, Cloud, AWS, DevOps, MSP, Field Tech, and Security-adjacent roles.
- **Remote-Friendly Resources:** Added and highlighted U.S. citizen-friendly remote/contractor boards (e.g., Dice, Gun.io, ClearanceJobs) and global remote networks (Otta, Himalayas).
- **Link Audit Automation:** Introduced a Python-based automated link auditing script (`scripts/audit_links.py`) to keep the repository healthy.
- **HTTPS Normalization & Redirect Cleanup:** Programmatically upgraded functional `http://` links to `https://` and resolved redirects to their final destinations.
- **Documentation Improvements:** Added dedicated files for filtering rules (`FILTERING_RULES.md`), search keywords (`AUSTRALIA_IT_SEARCH_KEYWORDS.md`), and remote eligibility notes (`REMOTE_ELIGIBILITY_NOTES.md`).

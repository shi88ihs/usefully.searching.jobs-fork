# Filtering Rules for This Fork

## Keep
- English-language resources only.
- Australia, APAC, worldwide, or explicitly remote-friendly roles that can be done from Australia.
- IT support, Linux, cloud, AWS, DevOps, infrastructure, MSP, field tech, security-adjacent IT, and automation roles.
- Freelance or contract platforms that accept global talent or clearly support remote work.
- U.S.-based boards only when the posting accepts remote international applicants, contractors, or worldwide candidates.

## Exclude
- Non-English-only boards.
- Region-locked boards outside the target geography.
- Dead, redirected, spammy, or low-confidence links.
- Unrelated industries.
- Listings that require local work authorisation, residency, or tax setup unless the posting explicitly says Australia-based applicants are allowed.

## Review Before Keeping
- If a site says "remote" but does not mention Australia, APAC, worldwide, or a compatible timezone, mark it as needs verification.
- If a site has strong role fit but unclear geography, keep it only in a note or under needs verification until confirmed.
- Prefer quality over quantity.
